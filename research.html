RECENT WORK

Dynamic Range Partitioning With Pointer Machines & Binary Search Trees.

This research presents DRP-PMBST, a data structure that combines pointer machines and binary search trees to efficiently manage and search large numeric ranges. By dynamically partitioning data and using separate BSTs for each sub-range, DRP-PMBST enhances search and insertion performance with an average time complexity of O(log(query_num) + log k). This structure offers improved scalability and efficiency for dynamic data management.

Fully Retroactive Confluent Data Structure

The Fully Retroactive Confluent Data Structure is an innovative solution for managing historical data. It enables retroactive modifications of past data without affecting future states and supports confluent merging of divergent versions into a coherent new version. This approach combines efficiency in handling updates, insertions, and deletions with robust version control, making it ideal for collaborative editing, version control, and historical data analysis.
